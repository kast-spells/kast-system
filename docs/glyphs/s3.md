# S3 Glyph

S3-compatible storage management.

## Templates

- `s3._s3` - S3 bucket operations
- `s3._seaweed` - SeaweedFS integration
- `s3._bucket` - Bucket management

## Generated Resources

S3-compatible storage CRDs.

## Examples

```yaml
glyphs:
  s3:
    - type: _bucket
      name: my-bucket
      size: 100Gi
```

## Testing

```bash
make glyphs s3
```
